# 📄 Text Analysis Module (Readability & QA)

## 👩‍💻 Author

Jyothirlatha

---

## 📌 Description

This module is a part of a larger **Text Summarization & Paraphrasing Project**.
It focuses on analyzing text readability and providing question-answering functionality from given content.

---

## ⚙️ Features

### 🔹 Readability Analysis

* Calculates readability scores:

  * Flesch Reading Ease
  * Gunning Fog Index
  * SMOG Index

### 🔹 Question Answering (QA)

* Extracts relevant answers from given text
* Helps users understand content quickly

---

## 📂 Files Included

* `readability.py` → Calculates readability metrics
* `pdf_qa.py` → Handles question-answering from text

---

## 🧠 Technologies Used

* Python
* NLP (Natural Language Processing)

---

## 🚀 Usage

1. Provide input text
2. Run readability analysis
3. Ask questions based on the text
4. Get meaningful insights and answers

---

## 🎯 Purpose

To improve text understanding by:

* Measuring readability
* Providing quick answers
